# 安装

安装步骤参照下述图文描述

双击setup.exe后弹出安装命令窗口...

显示安装界面、点击按钮「是」

配置安全更新界面点击「下一步」

选中『创建和配置数据库』后继续『下一步』

根据宿主机环境选择对应的类型后，继续「下一步」

选中「单实例数据库安装」，继续「下一步」

懒人推荐选择「典型安装」，继续「下一步」

配置安装路径以及全局数据库名及口令，继续【下一步】（若弹出“输入的口令不符合Oracle建议的标准...”警告,可忽略并点击「是」继续下一步）

点击「完成」按钮开始执行安装

等待安装完成...
